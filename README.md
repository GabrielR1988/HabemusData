# Informes del Automotor

Aplicación web en Flask para generar y gestionar informes sobre dominios de vehículos.

## Funcionalidades

- Registro e inicio de sesión
- Pedidos de informes por patente
- Generación manual de PDFs
- Almacenamiento en Supabase Storage
- Notificaciones por correo

## Tecnologías

- Python + Flask
- SQLite / PostgreSQL (Railway)
- Supabase
- Bootstrap
